# 编辑体验强化计划

## 背景

项目已从手写基础编辑区切换到 Tiptap 官方模板，但仍处于“可用而未收敛”的阶段，需要持续清理多余能力并强化写作体验。

## 当前目标

1. 继续优化正文排版密度
2. 稳定选区改写与应用逻辑
3. 减少模板中与产品目标无关的按钮和扩展

## 当前决策

- 官方模板保留在仓库中，先接入再裁剪
- 正文视觉优先级高于工具条花哨能力
- AI 面板继续由业务层控制，不迁移到模板内部
- 后端工作区是文档唯一真相；前端只保留编辑缓冲、选区与候选 diff 展示
- MVP 的人工编辑采用整文保存；AI 改写按 `sessionId + docPath + plain-text range` 发起
- AI 返回整篇 Markdown 候选版本，前端只负责审核和确认应用，不再自行保格式

## 待办

- 评估是否移除图片上传和部分高级文本能力
- 增加编辑器回归测试
- 清理不必要的模板样式覆盖

## 进度

- 已完成：模板接入、标题/行距/分割线收紧、正文宽度放大
- 已完成：后端临时工作区 API、远程文档读取/保存、workspace-backed rewrite candidate
- 进行中：模板裁剪与稳定性回归
