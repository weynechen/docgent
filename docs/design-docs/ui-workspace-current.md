# 当前工作台设计

## 总体布局

当前 UI 为三栏工作台：

- 左栏：文档列表与版本历史
- 中栏：标签栏、正文编辑区、底部状态栏
- 右栏：AI Chat、工具执行状态与输入区

## 已实现交互

1. 中栏顶部有单标签栏，显示当前打开文件名。
2. 左右栏支持拖拽调整宽度。
3. 左右栏支持折叠，恢复按钮放在标签栏左右两侧。
4. 右栏不显示默认说明卡片和快捷按钮组，只保留实际对话与输入。
5. 保存消息下沉到底部状态栏，避免遮挡正文。
6. 编辑器已切换到 Tiptap 官方 `simple-editor` 模板，并针对写作场景做了排版收紧。
7. 右栏交互方向已经从“单轮改写”切到“聊天优先”，后续会承载多轮历史、流式消息和工具过程。

## 当前视觉倾向

- 以浅色工作台为主
- 正文宽度已放大，减少大面积空白
- 标题、段落、列表和分割线都做了更紧凑的写作排版调整

## 已知限制

- 官方模板引入后工具条能力较强，但与产品自身 AI 面板尚未完全统一
- 编辑器模板源码量较大，后续需要提炼真正用到的能力
- 右栏当前仍偏向单次 run 结果展示，尚未完成真正的流式 chat 体验
- 无选区任务、工具调用可视化和 agent 自主循环仍处于待实现状态
